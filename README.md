# Jadlis Advisor — AJTBD

Интерактивный продуктовый advisor на основе методологии **Advanced Jobs To Be Done v13** (Иван Замесин).

## Что делает

Помогает принимать продуктовые решения по методологии AJTBD:

- **Граф работ** — построение и анализ иерархии работ пользователей
- **Сегментация** — поиск и выбор сегментов по работам (Core Jobs, Big Jobs, ABCD)
- **Создание ценности** — 17 механик создания ценности продукта
- **Запуск продукта** — PMF-градиент, RAT-фреймворк, пивоты
- **Продуктовая стратегия** — механики стратегии, локальный/глобальный оптимум
- **Маркетинговая упаковка** — лендинги, баннеры, скрипты продаж на основе AJTBD
- **Исследования** — глубинные интервью, UX-тесты, гайды v3.3
- **Развитие продукта** — механики роста, снижение оттока, ABCD-сегментация

## Как работает

1. **Загружает контекст** из Supabase (тир, потребности, цели, SWOT, привычки)
2. Собирает контекст: что за продукт, какая бизнес-задача, что уже сделано
3. Определяет путь: запуск / продажи / рост / дифференциация
4. Применяет мастер-алгоритм: бизнес-задача → механики → исследование → гипотезы → ранжирование
5. Даёт конкретные рекомендации с обоснованием из методологии
6. **Сохраняет предложения** в `advisor_proposals` для отложенного рассмотрения в NeedsCore Dashboard

## Требования

- **Supabase:** Таблица `advisor_proposals` должна существовать (создаётся миграцией из `05-advisors-framework`)
- **MCP:** `.mcp.json` конфигурирует Supabase MCP сервер автоматически
- **Переменные окружения:** `SUPABASE_ACCESS_TOKEN` должна быть установлена

Если Supabase недоступен, advisor работает в режиме чистых знаний (без автоматической загрузки контекста).

## Установка

```bash
claude --plugin-dir "/path/to/Jadlis-Advisor-AJTBD"
```

## Использование

Advisor активируется автоматически при продуктовых вопросах или вручную через `/jadlis-advisor-ajtbd:ajtbd`.

Примеры запросов:
- "Помоги выбрать сегмент для нового продукта"
- "Как вырастить конверсию на лендинге?"
- "Продукт не продаётся, что делать?"
- "Составь гайд для глубинного интервью"
- "Какую продуктовую стратегию выбрать?"
